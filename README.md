# BMI Calculator

A web-based BMI (Body Mass Index) calculator built with ASP.NET (C#), with an automated CI/CD pipeline using GitHub Actions and deploying to azure web apps.

## What it does

Takes height and weight input, calculates BMI, and returns the result with the corresponding category (underweight, normal, overweight, obese).

## Stack

- **Backend:** C# / ASP.NET
- **Frontend:** HTML, CSS, JavaScript
- **CI/CD:** GitHub Actions ([workflow](.github/workflows))
